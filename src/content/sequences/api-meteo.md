### 1. Contexte & Objectifs
Après avoir étudié les protocoles web (SNT) et le format JSON, cette séquence vise à appliquer ces concepts de manière concrète.

**Objectifs (Bulletin Officiel) :**
- Identifier les protocoles liés à une requête `http`.
- Utiliser une API REST documentée.
- Manipuler des données JSON en Python (dictionnaires imbriqués).

### 2. Déroulement
La séquence s'est déroulée sur 3 séances de 2h :
- **Leçon 1 :** Découverte de l'ordinateur : Rôles, familles et schéma fonctionnel
- **Leçon 2 :** Au cœur du traitement : Le CPU et l'architecture Von Neumann
- **Leçon 3 :** Les Mémoires (Vive, Morte et de Masse)
- **Leçon 4 :** Carte mère et périphériques
- **Leçon 5 :** Synthèse et Spécifications Techniques

### 3. Réflexion & Évaluation
L'aspect "données réelles" a été un facteur de motivation clé. L'évaluation (lien ci-dessus) n'était pas un QCM, mais un petit projet d'adaptation : "Comment modifier le script pour afficher le coucher du soleil (sunrise/sunset) ?"
